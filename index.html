Answer the questions in order to specify the following attributes for your new VM:

  1. CPU Count: Number of Logical Processors
  2. Memory Allocation: GB of Memory
  3. Disk Size: This is the size of the additional data disk requirements
  4. Location: Geographic location for the VM
  5. Environment: The logical Environment for the deployement
  6. Operating System: Base Operating System for the VM - Note no ssh access for windows VMs.

NOTE: All systems come with a minimal OS drive with infra tools installed

Backups are only available in the Production Environment.
